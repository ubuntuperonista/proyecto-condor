# Proyecto Cóndor

El Proyecto Cóndor es un proyecto de mejora de escenarios de simulación de vuelo de la Argentina utilizados por el programa de simulación de vuelo libre Flightgear (https://flightgear.org/).
Usa la biblioteca nativa de objetos 3D Shared para mejorar el detalle de aeropuertos, aeródromos, y corredores aéreos locales.

Project Condor is an Argentine scenery improvement project for the Flightgear free flight simulation software. It use the native Flightgear Terrasync and standard Shared model library to improve local airdromes, airports and air corridors.
